# 🔐 Secure DevSecOps Lab

Project demonstrating a basic **Shift Left Security pipeline**.

---

## 🚀 What this project shows

- CI/CD pipeline with GitHub Actions
- SAST scanning using Semgrep
- SQL Injection vulnerability simulation
- Secure coding fix using input validation
- Pull Request workflow

---

## 🧪 Security Flow

1. Vulnerable code created
2. SAST detects issue
3. Fix applied in branch
4. Pull Request created
5. Code merged to main

---

## 🛠 Tech Stack

- Node.js (Express)
- GitHub Actions
- Semgrep
- Git

---

## 📌 Concept

Shift Left Security applied in CI/CD pipeline.
